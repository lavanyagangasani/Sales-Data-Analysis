# Sales Data Analysis Using Python

## Project Overview

This project analyzes a sales dataset using Python to identify sales trends and generate useful insights. It includes data cleaning, sales analysis, and data visualization using charts.

## Features

- Read sales data from a CSV file.
- Clean the dataset by removing missing values and duplicate records.
- Calculate total, average, highest, and lowest sales.
- Analyze monthly sales trends.
- Visualize sales data using bar charts, line charts, and histograms.
- Generate a cleaned dataset and sales report.

## Technologies Used

- Python
- Pandas
- Matplotlib

## Dataset

The dataset contains the following columns:

- Date
- Sales

## Project Structure


## How to Run

1. Open the project in Google Colab or any Python IDE.
2. Upload the `sales_data.csv` file.
3. Run the `sales_analysis.py` file.
4. View the generated charts and sales report.

## Output

- Sales summary
- Monthly sales analysis
- Bar chart
- Line chart
- Histogram
- Cleaned dataset
- Sales report
